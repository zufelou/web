# 剑指基金交易网站

#### 介绍
{**以下是 Gitee 平台说明，您可以替换此简介**
为股民提供一个完整的基金交易功能，包括：登录注册用户，查询基金，买卖基金，添加自选股，风险评估，相关基金信息推送，根据用户反馈信息进行推送。相对于传统的基金交易网站，我们能够将庞大的数据进行可视化分析，让股民更加清晰的观察到自选股的走势，从而减少观望的时间。投资有风险，在数据可视化的分析下，可以对股民进行理财管理，做到风险最小化。}
